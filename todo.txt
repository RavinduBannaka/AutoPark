
You are required to help me **design, complete, and finalize an Android mobile application (Kotlin)** for a **Smart VIP Parking Facility**.

### Application Overview

The application supports **two types of users**:

1. **Admin (Parking Manager)**
2. **Vehicle Owner (Customer)**

The system uses:

* **QR code tokens** for parking access
* **Firebase real-time database** for data storage
* **JSON data handling**
* **Google Maps integration** for parking lot locations

---

### Admin (Parking Manager) – Features

The Admin manages the entire parking system.

**Required functionalities:**

* Register and manage vehicle owners
* Register and manage vehicle details
* Create and manage parking lots
* Define parking rates (normal, VIP, hourly, overnight)
* Scan QR codes for vehicle entry and exit
* Record vehicle time-in and time-out
* Automatically calculate parking charges
* Generate monthly parking reports
* Generate monthly invoices
* Calculate and manage overdue charges

---

### Vehicle Owner (Customer) – Features

The Vehicle Owner uses the parking facility and views personal parking data.

**Required functionalities:**

* View personal profile details
* View registered vehicle information
* Display QR code token for parking access
* View parking history (date, time-in, time-out)
* View parking charges
* View monthly invoices
* View overdue charges
* View parking lot details and location on Google Maps
